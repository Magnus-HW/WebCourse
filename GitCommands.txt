1) Create branch:
	git branch coconut

	git checkout coconut

3) Make changes/ work in the branch

4) Check for branch modification:
	git status
5) Add changes to staging directory:
	git add -A
5) Commit chancges:
	git commit -m "THE COCONUT HAS BEEN CRACKED"
6) Push branch to remote repository:
	git push -u origin coconut
7) Merge with master:
	git checkout master
	git pull origin master
	git merge coconut
8) Push changes to the remote repository
	git push origin master
9) Delete branch:
   git branch -d “branch name”             — delete from local
   git push origin —delete “branch name”   — delete from remote
